Current Weather

Project Summary: We used historical data from residential solar panels and weather data from OpenWeather. The first step of our project was to use a linear regression correlation matrix to determine what meteorological factors have the strongest correlation with solar power output. The meteorological factors that have the greatest influence on solar output were used to build a multivariate machine learning model to forecast power output based on weather forecasts.
